# 🛒 Customer Segmentation using K-Means Clustering

## 📌 Project Overview

This project focuses on **Customer Segmentation** using Machine Learning.
The goal is to group customers based on their purchasing behavior to help businesses make better marketing decisions.

Using the K-Means clustering algorithm, customers are segmented based on:

* Annual Income
* Spending Score

---

## 🎯 Objective

* Identify different types of customers
* Analyze spending patterns
* Help businesses target the right audience

---

## 📊 Dataset

* Dataset: Mall Customer Segmentation Dataset
* Features used:

  * Customer ID
  * Gender
  * Age
  * Annual Income (k$)
  * Spending Score (1–100)

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Applying K-Means Clustering
6. Elbow Method to find optimal clusters
7. Data Visualization
8. Business Insights

---

## 📉 Elbow Method

The Elbow Method is used to determine the optimal number of clusters by plotting WCSS (Within-Cluster Sum of Squares).

👉 Optimal number of clusters found: **5**

---

## 📈 Output

* Customers are divided into 5 clusters:

  * High Income – High Spending (VIP Customers)
  * High Income – Low Spending
  * Low Income – High Spending
  * Low Income – Low Spending
  * Average Customers

---

## 💡 Business Insights

* Target high spenders with premium offers
* Convert high-income low-spending customers
* Retain VIP customers with loyalty programs
* Focus marketing strategies based on customer segments
